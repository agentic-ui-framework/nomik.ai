Skills & Prompts

Your agents get sharper at your business.

A skill is a procedure your agents can follow: how to do one thing well, written down, with the gotchas learned the hard way. The fleet builds them as it works, and the proven ones spread to everyone.

Recipes vs skills

One is what you ask for. The other is how it's done well.

They work together. A recipe is the job; a skill is the craft an agent brings to it, and keeps improving.

The learning loop

Work once, get better forever.

Your fleet doesn't start from zero every morning. It captures what worked, proves it, and shares it, so the second time is always faster than the first.

1 · Do

An agent does the work

It runs the task, makes the calls, and hits the rough edges that only show up in your real data and your real tools.

2 · Record

It writes down what worked

The steps that landed, and the gotcha that bit it, become a skill: a short, named procedure with a clear "use this when" line.

3 · Prove

It marks it verified

A skill is only trusted once it has actually worked. Until then it's a draft, used with care, not gospel.

4 · Promote

The team inherits it

Proven skills get promoted to the shared canon, so every agent in the workspace starts the next job already knowing the trick.

What a skill looks like

Plain text, not a black box.

A skill is a short, readable file: a name, a one-line "use this when", the procedure, and the gotchas. You can read every one your fleet relies on, and so can the next agent.

  • Named and scoped: a sharp description tells an agent exactly when to reach for it.
  • Procedure, not prose: the steps to follow, in order, kept tight.
  • Gotchas: the failure that taught the lesson, dated, so it never repeats.
SKILL.md
---
name: reconcile-overdue-invoices
description: Use when chasing unpaid
  invoices across Stripe + QuickBooks.
verified: true
---
# Reconcile overdue invoices
1. Pull open invoices from both sources.
2. Match by invoice number, then amount.
3. Flag mismatches; never double-count.
4. Rank by days overdue, then amount.

## Gotchas
- Stripe lists amounts in cents.
  Convert before comparing to QuickBooks.
Shared canon and private memory

Lessons spread; context stays put.

Each agent keeps its own working memory. The skills that prove out get raised into a shared canon the whole workspace reads from, so a win on one desk lifts the whole team.

  • Private memory: an agent's own notes and drafts, scoped to its work.
  • Shared canon: the verified skills and standards every agent inherits.
  • Always yours: skills live in your workspace, readable and auditable, not locked in a model.
