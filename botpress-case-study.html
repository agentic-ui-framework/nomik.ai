Technical Solution: Conversational Payment Agents

1. Native Commerce Integration

Botpress chatbots gain built-in payment capabilities through Nomik's conversational commerce SDK.

const commerceBot = await nomik.kya.registerConversationalAgent({
  identity: `did:ai:botpress:commerce:${botId}`,
  platform: "botpress",
  capabilities: [
    "conversational_payment_processing",
    "cart_management",
    "subscription_handling",
    "refund_processing",
    "order_status_tracking"
  ],

  commerceSettings: {
    paymentMethods: ["card", "apple_pay", "google_pay", "crypto"],
    currencies: ["USD", "EUR", "GBP", "CAD"],
    maxTransactionAmount: 10000,
    autoRefundThreshold: 100
  }
});
