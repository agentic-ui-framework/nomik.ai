Developer Documentation

Quick Start

1. Install SDK

# Node.js / TypeScript
npm install @nomik/sdk
# Python
pip install nomik-sdk

2. Initialize Client

import { NomikClient } from '@nomik/sdk'

const nomik = new NomikClient({
  apiKey: process.env.NOMIK_API_KEY,
  environment: 'sandbox'
});

3. Execute Payment

const payment = await nomik.executePayment({
  agentId: 'my-agent',
  amount: 100,
  currency: 'EUR',
  recipient: 'recipient-id'
});
