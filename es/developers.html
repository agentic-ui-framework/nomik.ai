En la práctica

Una idea de cómo es.

REST limpio, una configuración MCP familiar y un conector de una sola fila. Todos los detalles, la autenticación y los códigos de error están en la documentación.

REST · listar los agentes de un espacio de trabajo
# Authenticated with a bearer token
curl https://api.nomik.ai/v1/workspaces/$WS/agents \
  -H "Authorization: Bearer $TOKEN"

# → every response shares one envelope
{
  "success": true,
  "data": [
    { "id": "agt_8f2", "name": "Fin", "role": "finance" }
  ],
  "metadata": { "requestId": "req_3a9" }
}
MCP · dar a los agentes una herramienta que tú alojas
// Mount any MCP server into the workspace.
// Its tools join the ones agents already have.
{
  "mcpServers": {
    "warehouse": {
      "command": "npx",
      "args": ["-y", "@acme/warehouse-mcp"],
      "env": { "REGION": "eu" }
    }
  }
}
Conector · una fila se convierte en una integración real
// No SDK to write. Declare it; the factory builds the actions,
// the vault holds the credential, and risk gates the writes.
export const acme = {
  kind: "acme",
  displayName: "Acme",
  category: "Operations",
  auth: "api_key",            // or "oauth"
  baseURL: "https://api.acme.com/v2",
  risk: "outbound",           // reads stay free; writes wait for approval
}

Nomik Desktop

El mismo SDK impulsa un agente local-first.

Nomik Desktop es un agente independiente que se ejecuta en tu máquina, con tu modelo y tus herramientas, seguro por defecto. Un connect y se conecta a un espacio de trabajo Nomik a través de este mismo SDK: manos locales, cerebro en el espacio de trabajo, todo bajo la validación humana.

  • Seguro por defecto: secretos cifrados, ejecución que prioriza el sandbox, herramientas con control según el riesgo, un registro de auditoría, activo desde la instalación.
  • Nativo del espacio de trabajo: gobernado por el mismo control, los mismos límites de gasto, el mismo RBAC y el mismo registro de actividad que tu flota ya usa.
  • Trae tus habilidades: importa tus habilidades de OpenClaw existentes, luego versiónalas y puntúalas.
Conoce Nomik Desktop →
nomik-desktop connect
# a local agent, one command from your workspace
$ nomik-desktop connect
✓ connected as "Nomik Desktop @ this-mac" · role: agent

# local hands the fleet can use, with approval
▸ files.read  ~/q3/report.md  →  ok
▸ shell.run  build.sh  →  waiting for your approval
# every action lands in the activity trail
